## Writing Your Own Tutorials

Add a `.json` file to `tutorials/` with this structure:

```json
{
  "title": "Tutorial Title",
  "book": "Optional source reference",
  "views": [
    {
      "title": "View Title",
      "text": "Explanatory text (plain text, pre-wrap monospace display)...",
      "group": { "type": "Zn", "n": 6 },
      "morph_target": { "type": "Zn", "n": 3 },
      "state": {
        "selected_1": "2",
        "selected_2": null,
        "show_subgroup": true,
        "show_conj": false,
        "show_table": false,
        "morph_mode": false,
        "morph_idx": 0
      }
    }
  ]
}
```

**`group.type`** / **`morph_target.type`** values: `"Zn"` (requires `"n"`), `"Dn"` (requires `"n"`), `"S3"`, `"V4"`.

**`selected_1`** / **`selected_2`**: element label string (e.g. `"2"` for Z‚āÜ, `"r"` for D‚āô, `"(12)"` for S‚āÉ) or `null`.

**`morph_mode`**: `true` to open the Morphism View, `false` for Cayley Diagram (default).

**`morph_target`**: the target group for the morphism diagram (optional; defaults to Z‚āô with n=6).

**`morph_idx`**: which homomorphism to display (0 = first/most interesting; isomorphisms sort before non-trivial maps, trivial map sorts last).

**Cube Lab state fields** (all optional; set `cube_active: true` to open the Cube Lab tab):

| Field | Type | Description |
|-------|------|-------------|
| `cube_active` | bool | Open the Cube Lab tab |
| `cube_mode` | string | `"conjugation"` \| `"commutator"` \| `"order"` \| `"parity"` \| `"generators"` |
| `cube_conj_seq` | string | Sequence for conjugation mode (e.g. `"R U R' U'"`) |
| `cube_conj_g` | string | Conjugating move `g` (e.g. `"F"`) |
| `cube_comm_g` | string | First move for commutator mode |
| `cube_comm_h` | string | Second move for commutator mode |
| `cube_order_seq` | string | Sequence for order mode |
| `cube_parity_seq` | string | Sequence for parity mode |
| `cube_gen_active` | list | Active face generators, e.g. `["U","D"]` |
